Vasikka Slack bot

This is a bot for reporting stuff

With this bot you can:

  1. Listen and respond to Slack slashcommands
  2. Check the slashcommand message for keywords (targets) listed in team-specific Google Sheet
  3. Save the message and metadata to team-specific Google Sheet
  4. Provide team-specific helpmessage for usage instructions
  5. Report on saved messages by target or all
